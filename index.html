Model 1

This is the first model I made. It had about 98% accuracy and I was super proud. It turns out real hand writing isn't like mnist. It isn't centered and scaled accordingly.

This model main problem is that large or small numbers don't work. Number that are not in the center work poorly too.

Model 2

Data agumentation was the solution. I shifted the numbers randomly and rotated them randomly. I was able to scale them up and down but that led to the line to be thicker. Since the drawn line is the same width the whole time it didn't work out.

The main problem with this is that small numbers work bad. Esacially if they are not in the center.

Model 3

This model is the third and final. It addresses alle the previous problems. It shifts and rotates randomly. It scales the numbers up and down while keeping the line width.

It has a problem with the number eight. If you draw number 0 and put a line through it the model should output 8 but it output 0.
